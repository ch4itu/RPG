# Starfall: Tiny Retro RPG

A fully offline, browser-playable retro RPG made with plain **HTML + CSS + JavaScript** and rendered on a pixel-art style `<canvas>`.

## Run

Open `index.html` directly in your browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Controls

- **Move:** WASD or Arrow Keys
- **Interact:** E or Space
- **Battle:** 1 (Attack), 2 (Star Skill), 3 (Potion)

## Story

A fallen star has corrupted the wilds around Oakvale. You are a wandering fighter summoned by Elder Rowan to recover the **Star Crystal** from the eastern cave, defeat awakened creatures, and restore peace.

## Features

- Tile-based open map (village, forest gate, cave)
- NPC dialogue and quest progression
- Random encounters in grass/cave tiles
- Turn-based battles with XP and level-ups
- Potions, stats, and quest HUD
- Pixel-art retro rendering with no external assets
